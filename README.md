# 📄 AI-Powered PDF Question Answering System

An AI-powered PDF Question Answering System built using Flask, Sentence Transformers, Semantic Search, Ollama, and Gemma 3.

The application allows users to upload a PDF document and ask natural language questions about its content. The system retrieves relevant document sections using semantic similarity and generates context-aware answers using a local Large Language Model (LLM).

---

## 🚀 Features

- Upload PDF documents
- Automatic PDF text extraction
- Intelligent document chunking
- Semantic search using embeddings
- Context-aware answer generation
- Suggested document-level questions
- Source page references
- Conversation history
- Recent questions section
- Remove uploaded PDF option
- Modern responsive UI
- Fully local AI inference using Ollama

---

## 🧠 AI Concepts Used

- Retrieval Augmented Generation (RAG)
- Semantic Search
- Embeddings
- Vector Similarity Search
- Prompt Engineering
- Large Language Models (LLMs)
- Context Retrieval
- Document Chunking
